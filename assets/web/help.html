Overview

Image Tool for Bing allows you to browse the recent history of "images of the day" hosted by the Bing search engine. Currently, Bing maintains a history of seven days worth of images and descriptions that the application lets you view, save, share, or set as your device wallpaper.

Features

By default, the main application page displays the current image of the day. The following describes in more detail the action bar items:

  • Previous - Go to the previous (older) image of the day.
  • Next - Go to the next (newer) image of the day.
  • Show Notes - Toggle display of the image description and copyright notice.
  • Copy Image Link - Copy image link URL to the BlackBerry device clipboard.
  • Save Image - Download and save the image to your device.
  • Share - Invoke the system share card to distribute image link URL via text, email, BBM, etc.
  • Set As Wallpaper - Invoke the picture editor card to edit and set the device wallpaper.
  • Refresh - Reload the current image of the day.

Settings

Choose Settings from the application menu and adjust the options listed below, as necessary:

  • Refresh image list at startup - Enables automatic downloading of the current image of the day when the application starts. Disable this if you want to manually refresh images.
  • Region / Language - Specifies your region and language, which the Bing search engine uses to customize the current image of the day.
  • Orientation - Specifies the size and orientation of images downloaded to your device. Choose a smaller image size if you wish to reduce network bandwidth (e.g., over your mobile network). Otherwise it is recommended to choose an image size equal to or larger than your device resolution, as follows:
    • BlackBerry Z10: 768x1280 (Portrait)
    • BlackBerry Z3, Z30: 720x1280 (Portrait)
    • BlackBerry Q5, Q10, Classic: 720x1280 (Portrait) or 1280x720 (Landscape) and use the Set As Wallpaper action to crop the image accordingly.
    • BlackBerry Passport: 1080x1920 (Portrait) or 1920x1080 (Landscape) and use the Set As Wallpaper action to crop the image accordingly.
